# Pandas-Dataframe-Lab

## Overview
This repository contains hands-on exercises for learning and practicing **Pandas DataFrame** functionalities in Python.  
The focus is on **data manipulation, exploration, and analysis** using the `flights` dataset from the Seaborn library.

The exercises include:
- Loading and exploring datasets
- Accessing and filtering data
- Basic statistics and descriptive analysis
- Sorting, grouping, and aggregation
- Handling missing values
- Adding, modifying, and renaming columns
- Merging, concatenating, pivoting, and crosstabs
- Reshaping and indexing
- Rolling and expanding calculations
- Sampling and string operations
- Correlation, covariance, masking, splicing, and datetime operations

## Dataset
- **Dataset used:** `flights` dataset from `seaborn`  
  Contains monthly number of airline passengers from 1949 to 1960.
